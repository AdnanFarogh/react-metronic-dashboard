## Installation

- Buy license of [Metronic theme](http://keenthemes.com/preview/metronic/)
- Copy Metronic *assets* folder to src/static
- Run install and start script

```
  npm install & start
```
## Source Generator
```
  npm generate
```

Supporting Types:
  - Components: Generate new Components `[Es6Class | Stateless]`
